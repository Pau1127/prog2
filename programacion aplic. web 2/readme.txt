Nombre del Proyecto: MandyEFV Samolo Osita - Explora Música, Anime y Tecnología

Desarrollado por: [Paula Alejandra Cid Cruz]

Descripción:
Este sistema web presenta una plataforma para explorar temas variados como música, videojuegos, anime y tecnología. La aplicación también permite ofrecer contenido adecuado para diferentes grupos de edad y tiene un banner condicional para aquellos usuarios que muestran interés en ciertos temas.

Archivo Inicial:
Inicia la página web abriendo el archivo `inicio.php`. Este archivo es el punto de entrada de la página principal.php y muestra gran parte del contenido incluyendo el menú de navegación, secciones temáticas y el contenido específico para mayores o menores de edad.

Instrucciones de Uso:
1. Abre `inicio.php` en un servidor que soporte PHP (por ejemplo, XAMPP, WAMP, MAMP, o un servidor web en línea).
2. Ingresa tu nombre y edad en el formulario inicial que reenviara a la página principal.php . Esto permitirá que el sistema determine si eres mayor o menor de edad, y cargue el contenido adecuado.
3. Si seleccionas la opción "Interesado" en el formulario, se mostrará un banner adicional con contenido especial.

Navegación y Funcionalidades:
- La barra de navegación te permite acceder a las diferentes secciones del sitio.
- Contiene dos menús, uno que te envía a la página Cuento.php y otro a la serie Fibonacci.php.
- El menú desplegable presenta enlaces directos a secciones específicas como música, videojuegos, anime, y tecnología.
- La página contiene un carrusel que muestra imágenes representativas de cada sección.
- Según tu edad, se carga contenido específico para mayores o menores de edad, incluyendo información relevante y segura para cada grupo.
- Si estás interesado en el contenido adicional, un banner especial aparecerá con imágenes promocionales.

Requisitos:
- PHP 
- Servidor web (local o en línea)
- Conexión a Internet para cargar librerías externas (Bootstrap)
